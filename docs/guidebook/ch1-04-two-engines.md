# 1.4 两种 AI 引擎——Claude 和 Pi

上一节我们认识了项目的技术栈，知道了大楼是用什么材料盖的。现在走进引擎室，看看是什么在驱动这栋大楼运转。

买车的时候，你可能会在汽油车和电动车之间纠结。两种车都能把你从 A 送到 B，但工作方式完全不同：汽油车烧油，靠内燃机；电动车充电，靠电池和电机。选择哪一种，取决于你更看重什么——是加油站的便利，还是充电桩的安静？

Craft Agents 的引擎室里也停着两台"发动机"：Claude Agent SDK 引擎和 Pi SDK 引擎。它们都能让 AI Agent 跑起来，但设计哲学和适用场景各不相同。

## Claude Agent SDK 引擎——专用充电桩

Claude Agent SDK 引擎是专门为 Anthropic 的 Claude 模型打造的。就像特斯拉的超充站只能充特斯拉，这台引擎天生就是给 Claude 用的，从底层 API 到交互协议都做了最原生的适配。

它的认证方式有两种：
- **Anthropic API Key（API 密钥）**：直接用 Anthropic 官方的密钥连接，最简单直接。
- **Claude Max/Pro OAuth（开放授权）**：如果你订阅了 Claude Max 或 Pro，可以直接用 OAuth 登录，不需要额外申请 API Key。

因为和主程序"住在一起"，Claude Agent SDK 引擎直接在主进程（Main Process）内运行。你可以把它想象成一台发动机直接装在车架上——没有传动轴的损耗，响应速度最快。

但这台引擎并不封闭。通过自定义 Base URL（基础地址），它也能接入其他服务：OpenRouter 可以用一个 API Key 访问上百个模型，Vercel AI Gateway 提供了请求路由和缓存，Ollama 则让你在本地跑开源模型。这些第三方端点走的仍然是 Claude Agent SDK 的通道，只不过目的地不再是 Anthropic 的服务器，而是你指定的地址。

## Pi SDK 引擎——万能充电器

Pi SDK 引擎走的是另一条路：多 Provider（提供者）设计。它不是专为一个 AI 品牌打造的，而是一个通用适配层，能对接十多个 AI 提供商。

目前 Pi SDK 支持的连接包括：
- Google AI Studio（Gemini 模型）
- ChatGPT Plus / Pro（通过 Codex OAuth）
- GitHub Copilot（通过 OAuth 设备码登录）
- OpenAI API Key
- Amazon Bedrock
- 以及更多……

像不像一个万能充电器？不管你手上是特斯拉、比亚迪还是蔚来，插上就能充。

Pi SDK 引擎还有一个关键的设计选择：它在独立的 Subprocess（子进程）中运行。主进程通过 JSONL 协议（每行一条 JSON 消息）和子进程通信。你可以把它想象成一台发动机装在独立的机房里，通过电缆把动力传过来——就算发动机出了问题，也不会波及主楼。这种隔离设计让系统更安全、更稳定。

此外，Pi SDK 原生支持一些高级功能：
- **Auto Compaction（自动上下文压缩）**：当对话变得很长时，自动压缩历史内容，避免超出模型的上下文窗口限制。
- **Provider 路由**：根据模型名称自动找到正确的 AI 提供商，你不需要手动指定"这个模型该走哪条路"。

## 为什么需要两种？

看到这里你可能会问：既然 Pi SDK 什么都能接，为什么不直接用 Pi SDK 就好了？

因为两台引擎解决的是不同的问题。

Claude Agent SDK 给你的是**最原生的 Claude 体验**。它和 Anthropic 的 API 深度绑定，工具调用的协议、流式响应的格式、权限控制的粒度，都是 Claude 一等公民该有的样子。如果你主要用 Claude 模型，这是最直接、最快速的选择。

Pi SDK 给你的是**多 Provider 灵活性**。当你的团队里有人用 Google AI Studio，有人用 ChatGPT，还有人用 GitHub Copilot——你不需要强迫大家统一，Pi SDK 能把它们都接进来。它的子进程隔离和自动压缩也是为多模型场景量身设计的。

不是重复，而是互补。

## 对你来说有区别吗？

没有。

这是最重要的一点：不管底层跑的是哪台引擎，你看到的都是同一个界面，操作方式完全一样。创建会话、发送消息、查看文件变更、切换权限模式——所有交互都不会因为引擎不同而改变。引擎的差异被封装在底层，你唯一需要做的就是在设置连接时选择对应的 Provider，然后一切照旧。

就像你开汽油车和电动车，方向盘和踏板的位置是一样的。你不需要知道发动机是什么类型，只需要知道——车能开，而且开起来一样顺手。
