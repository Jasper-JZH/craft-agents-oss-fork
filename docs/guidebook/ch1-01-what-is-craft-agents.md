# 1.1 Craft Agents 是什么？

想象你有一个助手，它不仅能听懂你说的话，还能真的动手帮你把事办了——帮你发邮件、整理项目进度、查数据库、写代码、管文件，甚至把好几件事串在一起自动完成。你只需要告诉它"帮我做什么"，它就自己想办法去完成。Craft Agents 就是这样一个助手。

## 从聊天机器人到 AI Agent：关键的一步

你可能已经用过 ChatGPT、Claude 这样的聊天机器人。它们很聪明，能回答问题、写文章、翻译语言，但有一个根本的局限：它们只能"说话"，不能"做事"。你问它"帮我查一下邮箱里有没有未读的重要邮件"，它只能告诉你怎么做，却没法真的打开你的邮箱去看。

AI Agent（智能体）打破了这条边界。Agent 不只是聊天，它还能使用工具。就像一个真人助手，既能跟你沟通，又能拿起电话、打开电脑、操作各种系统来完成任务。聊天机器人是一张嘴，Agent 是一双手加一张嘴。

## Agent 能做什么？

Craft Agents 中的 Agent 拥有广泛的核心能力：

- **连接外部服务**：Gmail、Google Calendar、Slack、GitHub、Linear、Notion……告诉它"帮我连上 Slack"，它会自己找到 MCP（Model Context Protocol，模型上下文协议）服务器或公开 API，读完文档，引导你配置凭证，一切就绪。
- **操作数据库**：不管是本地的 SQLite 还是远程的 Postgres，Agent 可以直接查询甚至修改数据。
- **执行代码**：运行脚本、安装依赖、编译项目——跟在终端里操作一样，但不用你自己敲命令。
- **管理文件**：读写文件、整理目录、批量重命名，文件系统就是它的工作台。
- **自动化工作流**：设置定时任务、监控事件触发动作——比如"每个工作日早上 9 点总结昨天的 GitHub issues"，或者"当某个 Session（会话）被标记为 urgent 时自动开始处理"。

## Craft Agents 的独特之处

市面上已经有一些 Agent 工具了，但 Craft Agents 走了一条不太一样的路。

**它是桌面应用，不是命令行工具。** 大多数 Agent 工具跑在终端里，面对的是黑色背景和闪烁的光标。Craft Agents 是一个原生桌面应用，有完整的图形界面、Session 管理、文件变更对比视图。你不需要记住命令行参数，点击、拖拽、快捷键就够了。

**一个 Session 可以同时使用多个 AI。** 你可以在同一个工作区里配置多个 LLM（大语言模型）连接——Anthropic Claude、Google Gemini、ChatGPT、GitHub Copilot，甚至通过 OpenRouter 接入几百个不同的模型。每个工作区可以设置默认模型，也可以随时切换。不同的任务用最合适的 AI，而不是被绑死在一个提供商上。

**能自动发现和配置 MCP 服务。** MCP 是让 AI Agent 连接外部工具的标准协议。在别的地方，配置 MCP 通常意味着手写 JSON 文件、查阅文档、调试连接。在 Craft Agents 里，你只需要说"帮我加上 Linear 作为数据源"，Agent 会自己搜索公开的 MCP 服务器、读取文档、引导你完成认证，整个过程像跟朋友聊天一样自然。如果你已经有了现成的 MCP 配置 JSON，直接粘贴进去也行。

**Skills（技能）让 Agent 更懂你的工作。** 你可以给 Agent 添加专门的技能指令，比如"代码审查时关注性能问题"或"写邮件时用正式语气"。技能按工作区存储，用 `@` 提及就能激活，甚至在对话中途也可以随时引入新的技能。

## 什么是"Agent Native"？

这是理解 Craft Agents 最关键的一个概念。

你一定见过那些后来"加了 AI 功能"的软件——原本是一个普通工具，后来在某个角落加了一个 AI 对话框，感觉像是贴上去的补丁。这不是 Agent Native。

Agent Native 的意思是：软件从一开始就是为 AI Agent 设计的。Agent 不是附加功能，而是软件的核心交互方式。在 Craft Agents 里，你不是在"使用一个工具，顺便有 AI 帮忙"，而是在"跟 AI 对话，AI 替你操作一切"。你描述目标，Agent 想办法完成；你遇到问题，Agent 调整策略。整个软件的架构——Session 管理、数据源连接、权限控制、自动化触发——都是围绕"AI 主动做事"这个理念构建的。

打个比方：普通软件像是给你一把锤子，AI 功能像是给锤子装了个电动马达；而 Agent Native 软件从一开始就是一台全自动的木工车间，你只需要说出想要什么家具。

这就是 Craft Agents 的定位——不是一个"带 AI 的编辑器"，而是一个原生的 Agent 工作平台。
